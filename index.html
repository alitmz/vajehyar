How it works

New dictionary searches and cloud AI feedback need internet access. Installed Android PWAs can receive shared words through the system Share menu. Saved words, context sentences, review progress, active-practice writing, weekly-test history, adaptive discovery history, AI feedback history, XP, streaks, and backups stay on this device. API keys are deliberately excluded from backups. Persian text uses the Vazirmatn typeface when available.
