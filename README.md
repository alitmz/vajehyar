# VajehYar v2.5 — IELTS Band 7 Vocabulary Bank

VajehYar is a private, installable vocabulary-learning PWA for personal English and IELTS preparation.

## New in v2.5

- **Expanded discovery bank:** 687 fully testable offline items.
- **Academic vocabulary:** 463 high-value academic and IELTS-topic words.
- **Collocations:** 130 standalone collocations plus 1426 collocation links attached to headwords.
- **Phrasal verbs:** 94 useful phrasal verbs, mainly for natural speaking and general comprehension.
- **Adaptive selection:** unseen items are prioritised; recently shown items are deprioritised; missed items return after a reasonable delay.
- **IELTS 7 adaptive level:** balanced mode focuses on B2 and introduces more C1 as discovery accuracy improves.
- **Content focus:** balanced, academic vocabulary, collocations, or phrasal verbs.
- **Topic focus:** education, environment, technology, health, society, economy, work, or research.
- **Richer question types:** definition, Persian meaning, sentence completion, collocation completion, and phrasal-particle completion.
- **Richer saved cards:** discovery items retain CEFR level, topic, item type, and collocations.
- **Backup support:** adaptive discovery history is included in export and restore.

## Vocabulary design

The bank is IELTS-focused rather than an official IELTS word list. Academic selections are informed by widely used academic-vocabulary principles, while collocations and phrasal verbs were curated for useful production and comprehension. Definitions, Persian glosses, and examples are bundled locally so weekly discovery remains available offline.

## Deploy on GitHub Pages

1. Extract the ZIP.
2. Upload every file and the `icons` folder to the root of the existing `vajehyar` repository.
3. Replace the old files and commit.
4. Wait for GitHub Pages deployment to become green.
5. Open `https://alitmz.github.io/vajehyar/?release=2.5.0` and refresh once.
6. Do not clear site storage; your library and progress are stored locally.
