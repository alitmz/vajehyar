# واژه‌یار — PWA

## انتشار در GitHub Pages
1. در GitHub یک Repository عمومی با نام `vajehyar` بسازید.
2. همه فایل‌های داخل این پوشه را در ریشه Repository آپلود کنید.
3. وارد `Settings > Pages` شوید.
4. Source را روی `Deploy from a branch` بگذارید.
5. Branch را `main` و Folder را `/ (root)` انتخاب و Save کنید.
6. آدرس برنامه: `https://USERNAME.github.io/vajehyar/`

## نصب روی گوشی
لینک را در Chrome باز کنید و از منوی سه‌نقطه `Install app` یا `Add to Home screen` را بزنید.

## نکته
برای جست‌وجوی کلمه جدید اینترنت لازم است؛ مرور کلمات ذخیره‌شده آفلاین است.
